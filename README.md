# Docker Node Cypress

Docker image to run Cypress tests.

## Usage

You can use this image in the following way:

```bash
docker run -it -v ${pwd}:/source wesleyschneider/node-cypress:latest bash
```

## Versions

- Node.js: lts-buster
- Oracle Client: 21.4
